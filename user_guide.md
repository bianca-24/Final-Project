# User Guide — ESG Insight Dashboard

## Overview

ESG Insight is a web-based dashboard that allows investors and analysts to explore Environmental, Social, and Governance (ESG) scores for companies in the financial sector. The scores are extracted from SEC annual filings using AI (Google Gemini) and stored in a PostgreSQL database.

---

## Accessing the Dashboard

### Option 1: Via URL (Recommended)
The dashboard is publicly hosted and accessible directly in your browser — no installation required.

Simply open:
```
http://91.99.170.56:8501
```

### Option 2: Run Locally (For Developers)

#### Requirements
Make sure you have the following installed:
- Python 3.8 or higher
- The following Python libraries:

```bash
pip install streamlit plotly pandas psycopg2-binary python-dotenv
```

#### Environment Setup

1. Copy the `.env.example` file and rename it to `.env`
2. Fill in your database credentials:
   ```
   DB_HOST=your_host
   DB_PORT=5432
   DB_NAME=your_database_name
   DB_USER=your_username
   DB_PASSWORD=your_password
   ```

#### Running the Dashboard

```bash
streamlit run Dashboard.py
```

The dashboard will open at `http://localhost:8501`.

## Features

### 1. Company Search
At the top of the page, use the **Search for a company** dropdown to select a company. You can type to filter the list.

### 2. Year Selector
Next to the search bar, use the **Year** dropdown to select the filing year you want to analyze.

### 3. ESG Score Cards
After selecting a company and year, four score cards are displayed:

| Card | Description |
|------|-------------|
| **Overall ESG** | The average of E, S, and G scores |
| **Environment** | Score for environmental practices |
| **Social** | Score for social responsibility |
| **Governance** | Score for corporate governance |

Scores are on a scale of **0–10** and color-coded:
- 🟢 **Green** — Score 8–10 (Strong)
- 🟡 **Olive** — Score 6–8 (Moderate)
- 🟠 **Orange** — Score 4–6 (Weak)
- 🔴 **Red** — Score 0–4 (Poor)

### 4. ESG Radar Chart
The radar chart shows the balance between Environment, Social, and Governance scores. A larger, more symmetrical shape indicates a well-balanced ESG profile.

### 5. Company vs Industry Average
The bar chart compares the selected company's E, S, G scores against the industry average across all companies for the same year. This helps identify where a company stands relative to its peers.

### 6. Sentiment Summary
A text summary generated by the AI model provides a qualitative analysis of the company's ESG performance, highlighting strengths and areas of concern.

### 7. ESG Company Rankings
A full rankings table at the bottom shows all companies sorted by their overall ESG score for the selected year, including individual E, S, and G scores.

## Understanding ESG

| Pillar | What it measures |
|--------|-----------------|
| **E — Environment** | Carbon emissions, energy use, climate risk management, environmental targets |
| **S — Social** | Employee welfare, diversity and inclusion, community impact, human rights |
| **G — Governance** | Board structure, executive compensation, transparency, anti-corruption policies |

---

## Data Sources

- **SEC EDGAR API** — Annual reports (10-K filings) from publicly listed US financial companies
- **Google Gemini API** — AI model used to score and summarize ESG content from reports
- **PostgreSQL** — Database storing all scores, summaries, and company information
